Library management system
Challenge consist:
1. API for CRUD of a books, managing title, author, isbn and description.
2. Ability to manage books through a web interface.
3. Ability to check-in and check-out a book.
4. Ability to track state changes for a book.
5. Report that contains the current state of all books.

This is project refers only the point two.
